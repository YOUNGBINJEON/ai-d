## 후기 남기는 법

1. 해당 저장소를 fork
2. fork된 본인 소유 저장소를 clone

  ```bash
  $ git clone __본인저장소!!__
  ```

4. 본인 이름으로 된 마크다운 파일 생성 후 후기 작성
5. 본인 소유 저장소로 push
6. github에서 PR 보내기
